Summary:
A simple Interface to read, write, edit floppys for the DCPU-16 in 0x10c.
The supported Fs is the FHAT-16. Copy of specis located in ./Copys Of docs.

At the moment only botable Floppys can be generated, Adding/removing of files and directorys is working.
Saving of the resulting floppy is working.
Some neat memory usage gauge is working.
The Endianess for FileIO operations can being selected. This effects only external IO operations, no internal operstions.

suitable releases are located in ./Releases

FHAT-16 Bootloader is located in ./BinExamples
